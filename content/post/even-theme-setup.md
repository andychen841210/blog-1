## Git submodule

To remove a submodule, [do](https://stackoverflow.com/questions/29850029/what-is-the-current-way-to-remove-a-git-submodule)

```bash
git submodule deinit <asubmodule>    
git rm <asubmodule>
rm -rf .git/modules/<asubmodule>
```
